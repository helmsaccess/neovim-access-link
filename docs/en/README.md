# English documentation

The English documentation is separated by audience:

1. [Quick Guide](manual/quick-guide.md) — the shortest path to a first local
   or remote connection.
2. [User manual](manual/README.md) — operation, settings, connections, Braille,
   and troubleshooting.
3. [Developer documentation](development/README.md) — architecture, protocol,
   security, compatibility, tests, and release process.

The documentation build creates three independent English HTML files. The
maintained sources remain Markdown.
