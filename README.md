# Neovim Access Link

Neovim Access Link is an NVDA add-on that makes Neovim editing accessible
through structured editor information. It supports command-line Neovim in
Windows Terminal, both locally on Windows and remotely over SSH to Linux,
including multiple tabs, windows, and tmux sessions. Editor modes, navigation,
editing, completion, and menus are reported through speech, with experimental
Braille support also included.

This is alpha-to-beta software. Not every feature has extensive practical
coverage. Braille has not been tested with a physical display, so
hardware-specific defects may remain undiscovered.

[Current GitHub release](https://github.com/helmsaccess/neovim-access-link/releases/tag/v0.95.1) —
release notes for 0.95.1: [English](docs/en/development/changelog.md#0951) or
[German](docs/de/development/changelog.md#0951).
